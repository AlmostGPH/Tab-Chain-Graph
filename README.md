# Tab-Chain-Graph
使用力导向图将标签页按照跳转关系绘制成图，方便在打开多个标签页的时候查询并回溯自己的浏览链路

> 灵感来自Obsidian

![](https://s2.loli.net/2025/02/27/ncGJWzTM4KOjIiZ.png)
